# Demo — conservation across coefficient substrates

> **Run:** `python examples/substrate/substrate_demo.py`

The same engine over different coefficient rings (one Euclidean-domain core):

- **the substrate spectrum** — the same data read over ℚ, over ℤ (torsion), and
  over several 𝔽ₚ, tied together by one identity;
- **parametric conservation over ℚ[t]** — laws as a function of a parameter, with
  the exact parameter values where conservation degenerates (roots of the
  invariant-factor polynomials).
